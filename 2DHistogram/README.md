# 2D Histogram - Kassationsanalyse

## Formål

Formålet med programet er interaktivt at lave 2D histogrammer som kan benyttes til visualisering og analyse af datasættet.


## Opsætning

For at programmet virker skal følgende programmer være installeret:

* numpy
* pandas
* scipy
* matplotlib
* pyarrw
* openpyxl
* PyQt5
